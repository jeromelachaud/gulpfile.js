# microFramework [![Build Status][travis-image]][travis-url]
Nothing fancy, just the basic stuff.
- Jade
- Scss
- Bootstrap
- Normalize
- jQuery
- BrowserSync

Powered by Gulp

## usage
- run "gulp --dev" for developement
- run "gulp --production" for production (no sourcemaps + minify + uglify)
